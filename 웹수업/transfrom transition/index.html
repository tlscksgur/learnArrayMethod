HTML CSS JS

HTML은 웹 페이지의 구조를 만드는 언어입니다. 태그를 사용해 제목, 문단, 이미지, 링크 같은 요소를 표시합니다. 웹 페이지의 뼈대를 구성하는 역할을 합니다.

CSS는 웹 페이지의 디자인과 레이아웃을 꾸미는 언어입니다. 글자 색, 크기, 배경, 배치 등을 설정해 웹 페이지를 더 보기 좋게 만듭니다. HTML이 뼈대라면, CSS는 그 뼈대를 꾸미는 도구입니다.

JavaScript(JS)는 웹 페이지에 동작을 추가하는 언어입니다. 버튼 클릭, 애니메이션 같은 상호작용을 만듭니다. 웹 페이지를 더 동적으로 만듭니다.
